# SkillNest Server – Backend API for Skill Sharing Platform

This is the Express.js server for SkillNest, a community-driven skill-sharing platform. It handles routing, database operations, and API endpoints for managing users, courses, instructors, and enrollments.

## Technologies Used

- **Express.js** – Fast and minimalist web framework for Node.js
- **MongoDB** – NoSQL database for storing application data
- **dotenv** – Environment variable management
- **CORS** – Cross-origin resource sharing configuration
